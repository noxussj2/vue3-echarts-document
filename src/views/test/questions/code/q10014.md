# 如何实现小于 12px 的字体？



定位：初级、中级前端职位

概率：★★★

相关链接：[实现小于 12px 的字体](/practice/code/11002.html)

一句话回答面试官：浏览器默认的最小字号是 12px，可以通过 zoom 或者 transform scale 的方式实现。其中两者最主要的区别是 zoom 的缩放会让其占据的空间发生变化，而 transform scale 却不会。另外 transform scale 默认是相对于元素的中间部分进行缩放的。
