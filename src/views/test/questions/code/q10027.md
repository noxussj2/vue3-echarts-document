# 什么是防抖和节流？



定位：初级、中级前端职位

概率：★★★★★

相关链接：[防抖和节流](/documents/part2/javascript-advanced/throttle.html)

一句话回答面试官：防抖和节流一般都是在优化用户体验上会用到，例如防抖可以让用户多次点击一个提交按钮时，只会触发最后一次点击。而节流则是在规定时间内只能触发第一次事件，后续的触发将会被忽略，直到计时结束。
