# es6 新特性有哪些？



定位：初级、中级前端职位

概率：★★★★★

相关链接：[ES6 新特性](/documents/part2/es6/features.html)

一句话回答面试官：比较常用的特性有块级作用域 let const、解构赋值、箭头函数、promise、async await、class 类、ESM 模块化。还新增了数据结构 Map 和 Set，以及数组和对象的一些扩展方法。

较大可能性会二次提问 var let const 三者的区别。
