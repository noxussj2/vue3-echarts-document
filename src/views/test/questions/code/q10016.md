# 基本数据类型和引用数据类型有什么区别？



定位：初级、中级前端职位

概率：★★★★

相关链接：[以上两种数据类型的区别](/documents/part2/javascript-base/data-type.html#以上两种数据类型的区别)

一句话回答面试官：基本数据类型和引用数据类型的区别在于它们在内存中存储的方式不同，基本数据类型的数据是存储在栈中，而引用类型的数据是存储在堆中。引用类型在栈中存储的只是一个指针，最终指向堆中的数据。
