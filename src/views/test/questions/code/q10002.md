# 什么是 Web 标准以及 W3C 规范？

定位：初级、中级前端职位

概率：★★★

相关链接：[Web 标准 & W3C 规范](/documents/part1/html5/web.html)

一句话回答面试官：Web 标准就是指网页中结构、表现、行为三部分，分别对应 html、css、js。而 W3C 则是对这三者进行了约束规范，例如标签字母要小写，不允许随意嵌套等。

:::warning 注意
建议了解一下 W3C 比较常见的规范，面试官可能会让你罗列出来。
:::
