# 什么是 Promise？

定位：初级、中级前端职位

概率：★★★★★

相关链接：[ES6 新特性-基础使用](/documents/part2/es6/features.html#基础使用)

一句话回答面试官：Promise 是一个异步的实现方案，通常用来延迟返回函数结果。promise 主要有三种状态，分别是等待、成功、失败。可以通过 .then .catch 来监听异步是成功还是是失败。
