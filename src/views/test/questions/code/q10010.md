# 什么是 flex 布局？



定位：初级、中级前端职位

概率：★★★★

相关链接：[flex 布局](/documents/part1/css3/flex.html)

一句话回答面试官：flex 是目前比较流行的一种布局，因为它十分简单灵活，可以轻松实现大多数网页布局。主要包含了 6 个容器的属性以及 6 个项目属性，例如容器的 justify-content 属性可以实现两端对齐效果等等。
或者容器的 flex-direction 属性可以改变主轴的方向。
