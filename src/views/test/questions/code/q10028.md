# 什么是高阶函数？



定位：中级前端职位

概率：★★★

相关链接：[高阶函数](/documents/part2/javascript-advanced/senior.html)

一句话回答面试官：有两种情况都可以被定义为高阶函数，第一种是把函数作为参数传递给另外一个函数，第二种是把函数作为另一个函数的返回结果。使用高阶函数可以让这个函数变得更加灵活/也利于封装通用的功能。例如平时数组用到 forEach、map、sort 方法也是高阶函数。
