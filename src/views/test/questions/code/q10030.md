# 什么是数据结构&算法？



定位：高级前端职位

概率：★★★

相关链接：[数据结构 & 算法](/documents/part2/javascript-senior/introduce.html)

一句话回答面试官：常见的数据结构有数组、栈、队列、链表、集合、字典等，而常见的算法有深/广度优先搜索、冒泡排序这些。算法在我们程序当中是最重要的一个部分，指的是你写的代码的质量。就例如你做一道菜，数据结构就相当于锅碗瓢盆，而算法就是食谱，你做出来的菜好不好吃取决于食谱。
